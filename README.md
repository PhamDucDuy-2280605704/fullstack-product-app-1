\# 📦 Fullstack Product Management App



> Một ứng dụng quản lý sản phẩm hoàn chỉnh với \*\*NestJS\*\* backend, \*\*PostgreSQL\*\* database, và \*\*React\*\* frontend.



\## 🚀 Tính năng



\- ✅ Thêm, sửa, xóa sản phẩm

\- ✅ Phân trang (pagination)

\- ✅ Thông báo toast đẹp mắt

\- ✅ API RESTful có validation

\- ✅ Kết nối database PostgreSQL



\## 🛠 Công nghệ sử dụng



\*\*Backend\*\*  

\- NestJS

\- TypeORM

\- PostgreSQL

\- class-validator



\*\*Frontend\*\*  

\- React

\- Axios

\- React Toastify

\- CSS Modules



\*\*Deployment\*\*  

\- Render (Backend + DB)

\- Vercel (Frontend)
